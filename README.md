# Fundamental AutoCAD Drafting Exercises

A set of introductory AutoCAD exercises focused on practicing core drafting tools: line creation, geometric shapes, and dimensioning techniques. Developed in two parts as skills progressed.

## Contents
- Part A: Basic geometric constructions and dimensioning practice
- Part B: Continuation of Part A, reinforcing precision drafting techniques

## Files
- `portfolio-2a.dwg` / `portfolio-2a.pdf` — Part A
- `portfolio-2b.dwg` / `portfolio-2b.pdf` — Part B
